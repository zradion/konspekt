img[title="portrait"] {} - для картинок у которых title="portrait"
a[href^="htto://123"] {color: red;} - ссылка начинается с 123
a[href$=".jpg"] {color: blue} - ссылка заканчивается на .jpg

div > p - только для дочерних p
h1 ~ p - для всех сестринских p
div p + span - для span, который идёт после тега p
.item.selected {} - в классе item к элементу добавляем ещё класс (расширение)

Градиент:

background: #ddd; - сначала для браузеров не поддерживающих градиенты.
background: linear-gradient(90deg(to bottom,to top), start color, last color) - изм цвета
linear-gradient(red, yellow) - переход от красного к желтому сверху вниз.
linear-gradient(to right, red, yellow); - слева направо.
linear-gradient(to bottom right, red, yellow); - верх лев к низ прав.
можно задавать несколько цветов и под углом.
repeating-linear-gradient(45deg, red, yellow 7%, green 10%); - повторяющийся градиент с неравномерным распостранением.

background: radial-gradient(размер фигуры в позиции, start, last); - по умолчанию эллипс, размер - самый дальний угол, позиция - центр.
radial-gradient(closest-side at 60% 55%, red, black); - размер градиента
farthest-side(самая дальняя сторона)
closest-coner(ближайший угол)
farthest-coner(самый дальний угол)
repeating-radial-gradient()

background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...); - конический градиент (вокруг центра)
для создания круговых диаграмм добаввим border-radius: 50%; для цветов задаём начало и конец.

Граница, контур (border, outline):

border-style: solid; (dotted, dashed, double, groove, ridge, inset, outset, none, hidden)
border-width: 1px; (thin, medium, thick, em ...)
четыре значения - верх, право, низ, лево; три - верх, право и лево, низ; два значения - верх и низ, право и лево; одно - все направления
border-color: black;(rgb, hex, hsl, transporent) - тоже до четырёх значений
border-top-style, border-right-stile, border-bottom-style, border-left-style

border-style: dotted solid; - границы верх и низ, право и лево.
border-bottom: 5px solid red; - нижняя граница (width style color).

border-radius: px | % - закругление границ.
Одно или четыре знач. - все углы (с верхн лев), два - углы по диогонали, три - верх лев, следующий и подиогонали, прав нижн.

outline - рисуется за пределами границы элемента и не связан с общей шириной.
Св-ва: outline-style, outline-color, outline-width, outline-offset (добавляет пробел между контуром и границей), outline
p {margin: 30px; border: 1px solid black; outline: 1px solid red;
outline-offset: 15px;}

Margin, padding:

margin-top (right, bottom, left) - внешний отступ. Может иметь св-ва: auto - браузер вычисляет поля, размер в px или др., % - проценты, inherit - унаследуется от родителя.
четыре значения - верх, право, низ, лево; три - верх, право и лево, низ; два значения - верх и низ, право и лево; одно - все направления
margin: 0 0 10px 0; - только нижнее поле
Верхние и нижние поля у соседних элементов иногда сворачиваются в одно равное большему.

padding-top (right, bottom, left) - внутренний отсуп. Все св-ва: размер (в px и др.), % - процент, inherit - унаследуется от родителя
так же может иметь до четырёх значений.
box-sizing: border-box; - св-во позволяет задавать ширину вместе с padding.

Ширина, высота:

width, height могут иметь значения auto, length (px и др), %, initial (значение по умолчанию), inherit (наследуется от родителя).
общая ширина = width + padding + border
max-width - максимальная ширина элемента
min-width - при уменьшении окна появится горизонтальная прокрутка.
@media screen and (min-width: 1200px) {} - адаптивная настройка (медиа запрос) ширины контента для ПК.
