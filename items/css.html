Горячие клавиши для vs code:

Ctrl + Shift + P - палитра команд
Wrap with Abbrreviation - выбрать команду, тег в который нужно обернуть код
Ctrl + F - строка поиска
Ctrl + Shift + L - выделить выбр.знач; закр строку поиска;удалять или изменять
Ctrl + Shift + K - удалить выбранные строки
Ctrl + L - удалить строку со всем содержимым
Ctrl + Z - отменить действие
Ctrl + / - закоментировать (/* ... */)
Ctrl + (+-)(0) - увеличить массштаб (вернуть по умолчанию)
Ctrl + [(]) - сдвигать выделенный блок

Alt + Z - сделать текст читабельным
Alt + стрелка вверх(вниз) - переместить выбранную строку, блок

Селекторы:

* {} - для всех тегов
.className {} - для элементов с классом className
#idName {} - для элементов с id="idName"
h1, p, span {} - для перечисленных элементов
img[title] {} - для картинок с атрибутом title
img[title="portrait"] {} - для картинок у которых title="portrait"
a[href^="htto://123"] {color: red;} - ссылка начинается с 123
a[href$=".jpg"] {color: blue} - ссылка заканчивается на .jpg
[title*="брауз"] {} - для элементов у которых атрибут title содержит часть фразы "брауз"
div p - для всех p вложеных в div
div > p - только для дочерних p
div p + span - для span, который идёт после тега p
.item.selected {} - в классе item к элементу добавляем ещё класс (расширение)
ПСЕВДОКЛАССЫ:
a:link - обычная ссылка
a:visited {color: grey;} - после посещения ссылки она меняет цвет
a:hover{text-dtcoration: underline;} - действие при наведении курсора
a:active{color: blue;} - действие при нажатие клавиши мыши
a:focus - фокусировка на элементе
для нескольких состояний последовательнось важна
div p:first-child {} - из всех вложенных p взять первый
(:first-letter, :first-line, :befor, :after)
h3:before {content: ' [ '}
h3:after {content: ' ] '} - текст в этом теге взяли в [ ]
ПСЕВДОЭЛЕМЕНТЫ:
::after {content: ''};
::before {content: ''};
::selection - выделение пользователем текста
::first-letter - первая буква
::first-line - первая строка

a {text-decoration: none;} - удаляем подчёркивание ссылок
a:hover {text-decoration: underline;}
