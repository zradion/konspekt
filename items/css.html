Селекторы:

* {} - для всех тегов
.className {} - для элементов с классом className
#idName {} - для элементов с id="idName"
h1, p, span {} - для перечисленных элементов
img[title] {} - для картинок с атрибутом title
img[title="portrait"] {} - для картинок у которых title="portrait"
a[href^="htto://123"] {color: red;} - ссылка начинается с 123
a[href$=".jpg"] {color: blue} - ссылка заканчивается на .jpg
[title*="брауз"] {} - для элементов у которых атрибут title содержит часть фразы "брауз"
div p - для всех p вложеных в div
div > p - только для дочерних p
div p + span - для span, который идёт после тега p
.item.selected {} - в классе item к элементу добавляем ещё класс (расширение)

Margin, padding:

margin-top (right, bottom, left) - внешний отступ. Может иметь св-ва: auto - браузер вычисляет поля, размер в px или др., % - проценты, inherit - унаследуется от родителя.
четыре значения - верх, право, низ, лево; три - верх, право и лево, низ; два значения - верх и низ, право и лево; одно - все направления
margin: 0 0 10px 0; - только нижнее поле
Верхние и нижние поля у соседних элементов иногда сворачиваются в одно равное большему.

padding-top (right, bottom, left) - внутренний отсуп. Все св-ва: размер (в px и др.), % - процент, inherit - унаследуется от родителя
так же может иметь до четырёх значений.
box-sizing: border-box; - св-во позволяет задавать ширину вместе с padding.

Ширина, высота:

width, height могут иметь значения auto, length (px и др), %, initial (значение по умолчанию), inherit (наследуется от родителя).
общая ширина = width + padding + border
max-width - максимальная ширина элемента
min-width - при уменьшении окна появится горизонтальная прокрутка.

Текст, шрифт:

font-style: italic;(oblique, normal, inherit)
font-size: 16px; (1em = 16px - размер по умолчанию)
font-weight: 500; (normal, bold) 1vw - 1% ширины области просмотра.
font-variant: small-caps; - отобразить капителью (строчные буквы как прописные, только маленькие)
font: normal 16px 'Roboto', sans-serif;

text-indent: 20px; - красная строка.
text-align: center; (left, right, justify - строки подгоняются по ширине)
vertical-align: baseline; (text-top, text-bottom, sub, super)
text-decoration: none; (underline, overline, line-through, blink, inherit)
text-decoration-line (-color, -style, -thickness) можно несколько стилей в строке
p {text-decoration: underline red double 1px;}

text-transform: uppercase; (lowercase, capitalize)
letter-spacing: 5px; - расстояние между знаками.
word-spacing: 10px; - расстояние между словами.
line-height: 5px; - интерлиньяж (межстрочный).
text-shadow: 2px 3px 4px #4c7; (X, Y, радиус размытия, цвет) можно несколько теней в строке.
direction: ltr; (rtl, inherit) - направление текста.
