Изображения:

Технически тег <img> - создаёт удерживающее пространство.
атрибуты src и alt - обязательные (путь и альтернативный текст)
нужно указывать ширину и высоту для правильной загрузки width="40px" height="40px"
предпочтительней с атрибутом style="width:40px;height:40px;"
изображение ссылка:
  <a href="#"><img src="smile.gif" alt="smile"></a>
style="float: right;"(left) - внутри параграфа картика справа (слева)
align="bottom"(left, middle, right, top)
border="0" - без рамки
width, height, hspace, vspace - ширина, высоста, отступ гориз, верт.

APNG (.apng) - анимированая портативная сетевая графика
GIF (.gif) - формат обмена графическими данными
ICO (.ico, .cur) - Microsoft значок
JPEG (.jpg, .jpeg, .jfif, .pjpeg, .pjp) - фото изображение
PNG (.png) - переносимая сетевая графика
SVG (.svg) - масштабируемая векторная графика

<map> определяет изображение-карту. (с кликабельными областями)
<area> - определяет активную область внутри изображения
< picture> - определяет контейнер для нескольких ресурсов

В <img> добавляется атрибут usemap="#workmap" с # и именем карты.
  <map name="workmap">
    <area shape="rect" coords="34, 44,234,350" alt="Деталь" href="url">
    <area shape="circle" coords="300, 400, 50" onclick="myFunction()">
  </map>
  <script>
    function myFunction() {alert("текст при клике на указаный участок");}
  </script>
атрибут shape:
"rect" - прямоугольник, coords - x,y - верхний левый угол и нижний правый
"curcle" - круг, coords - x,y - центр круга, радиус в px
"poly" - полигональная область (много точек) coords="x,y,x,y,x,y"
"default" - вся область

Фоновое изображение на странице:
  <style>
    body {background-image: url('img/home.png')}
  </style>
Элемент <picture> содержит ряд элементов <source> и последнии элеметн <img>. Выводится изображение, которое соовтвтствует условию или последнее. 
  <picture>
    <source media="(min-width: 650px)" srcset="img_food.jpg">
    <source media="(min-width: 450px)" srcset="img_car.jpg">
    <img src="img_girl.jpg">
  </picture>

Таблицы:

<table> - контейнер для элементов табл.
<tr> - table row - вмещает содержимое строки
<th> - table heading - ячейка заголовка табл.(жирно и по центру)
<td> - table data - обычная ячейка(текст, числа, картинки..)
По умолчанию без границ. 
  table, th, td {border: 1px solid black;}
  table {border-collapse: collapse;} - убрать двойную рамку
colspan="2" - атрибут td(th) - захватывает ячейки по горизонтали
rowspan="2" - атрибут td(th) - захватывает ячейки по вертикали
align="left | center | right" - выравнивание для таблицы (обтекание)
valigh="top | middle | bottom" - выравнивание внутри ячеек
bgcolor="color" - цвет фона
border, width, tabindex="1" ..перключение между элементами
CSS: td {width: 100%; padding: 15px; text-align: left;}
table {border-spacing: 5px;} - между ячейками
tr:nth-child(even) {background-color: #ddd;} - чётные строки
<caption> - Название таблицы. Располагается внутри table.
<thead> - верхняя часть таблицы
<tbody> - тело. Может быть несколько.
<tfoot> - нижняя часть табл. В коде надо распологать после thead. В длинных таблицах thead and tfoot выводятся сразу; tbody может прокручиватся.
